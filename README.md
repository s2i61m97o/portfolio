# Portfolio Site

A personal portfolio website for **Matt Simpson**, a Frontend Developer skilled in building interactive, responsive websites. Built to showcase my projects, skills, and contact information.

## 🚀 Features

- **Modern Tech Stack**: Built with Next.js, React 19, and TypeScript.
- **Responsive Design**: Tailored experiences for mobile, tablet, and desktop viewports.
- **Styling**: Utilizes SASS/SCSS modules for clean, maintainable, and scoped styling.
- **Form Integration**: Contact form powered by Formspree.
- **Accessible & Semantic**: Uses HTML5 semantic tags and ARIA best practices.

## 🛠️ Built With

- Next.js 16
- React 19
- TypeScript
- SASS / CSS Modules

## 📁 Projects Featured

- **Space Tourism**: A mock, multi-page website with information about Space Travel.
- **Photosnap**: A multi-page platform for visual storytelling.
- **Weather App**: A real-time weather application using the open-meteo API.
- **Assembly: Endgame**: A hangman-style game featuring CSS transitions and React state management.

## 💻 Getting Started

To run this project locally, follow these steps:

1. Clone this repository:

   ```bash
   git clone https://github.com/s2i61m97o/portfolio.git
   ```

2. Navigate to the project directory:

   ```bash
   cd portfolio
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the development server:

   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
